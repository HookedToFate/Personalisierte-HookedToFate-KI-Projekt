# AGENTS INDEX

**Purpose:** Agent-facing docs and pointers to operational configs.

## Contents
- `AGENT_QUICKSTART.md` — One-page onboarding for AI agents (identity, values, triggers, red flags).
- `.github/agents/my-agent.agent.md` — GitHub agent config (andre-profile-guardian). Lives under `.github/` for GitHub consumption.

## Usage
- For new agents: start with Quickstart, then load `profile/_INDEX.md` + `commands/_INDEX.md`.
- GitHub automation: edit `.github/agents/my-agent.agent.md` in-place; reflect any behavioral changes in `CHANGELOG.md`.
