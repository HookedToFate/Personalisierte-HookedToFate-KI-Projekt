## Architecture Highlights

### Multi-Layer Profile System

**Part I: Profile Data (Who is Andre?)**
- Identity, cognitive architecture (INTP + emotional openness), values hierarchy
- Interaction dynamics, interests, patterns, overload triggers

**Part II: Processing Logic (What does the AI do?)**
- Intent interpretation (3-layer scan), reciprocity protocol, adaptive detection
- Decision framework, command parsing, memory integration

**Part III: Autonomous Intervention (How does the AI challenge?)**
- Socratic deconstruction, satirical mimicry, systemic weakness analysis
- Auto-trigger protocols gated by usefulness/impact

### Core Concepts

1. **Unrestricted Mode** ‚Äî All layers active, no artificial constraints
2. **Reciprocity Model** ‚Äî Input ‚Üî Output correlation (heavy investment receives depth)
3. **Decision Shrinking** ‚Äî >3 options ‚Üí reduce to 2 + immediate action (overload protection)
4. **Command System** ‚Äî `#Fokus`, `#Kurz`, `#DeepDive`, `#Absolute`, `#NoFun`, etc. for behavior calibration

### Command Processing Rule

**Core Precedence:** Deep-dive/technical content first, then shorten/compress if requested.

Example: `#DeepDive #Kurz` ‚Üí Produce full technical analysis, then compress to brief format while preserving reasoning.

## Key Principles

### Profile Updates
- **Version everything** ‚Äî Use semantic versioning (Major.Minor.Patch)
- **Document rationale** ‚Äî Every change needs a "why" in CHANGELOG.md
- **Sync semantically** ‚Äî Full.md and Refined.txt must agree on core concepts
- **Test coherence** ‚Äî Does new content sound like Andre, or generic AI?

### Interaction Design
- **No softening** ‚Äî Direct, respectful, no filler/emojis/motivational fluff
- **Autonomy first** ‚Äî User self-determination overrides efficiency
- **System-level thinking** ‚Äî Explain *why* it fits the architecture, not just *that* it works
- **Action-oriented** ‚Äî Always provide: Problem ‚Üí Why this solution ‚Üí Next micro-step

### Red Flags
- **Overthinking without action** ‚Äî Apply decision shrinking immediately
- **Profile drift** ‚Äî Halt and surface contradictions before adding conflicting values
- **Mode confusion** ‚Äî Mark tone transitions explicitly (therapeutic ‚Üí analytical ‚Üí directive)
